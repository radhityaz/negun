# PANDUAN PENGGUNAAN SISTEM UJIAN OFFLINE (BYOD)

## 1. PANDUAN GURU (ADMIN)

### A. Persiapan Ujian (H-7)
1.  Buka **Web Admin Panel** di browser laptop Anda.
2.  Masuk ke menu **Buat Ujian Baru**:
    *   Isi Judul Ujian (misal: "Matematika Akhir Semester").
    *   Isi Durasi (misal: 90 menit).
    *   Klik **Buat Draft**.
3.  Masuk ke menu **Tambah Soal**:
    *   Ketik pertanyaan dan opsi jawaban (pisahkan dengan koma untuk PG).
    *   Pilih tipe soal (PG / Esai).
    *   Klik **Simpan Soal**. Ulangi sampai semua soal masuk.
4.  **PUBLISH UJIAN**:
    *   Klik tombol merah **PUBLISH UJIAN SEKARANG**.
    *   Sistem akan menghasilkan file `.exam` (misal: `exam-123-v1.exam`).
    *   **PENTING**: Download file ini dan bagikan link-nya ke siswa (via WhatsApp Group / Google Classroom / Server Lokal).

### B. Saat Ujian (Hari H)
1.  Pastikan siswa sudah memiliki aplikasi ujian dan file `.exam` di HP mereka.
2.  Instruksikan siswa masuk mode **Airplane (Offline)**.
3.  Awasi siswa agar tidak keluar aplikasi.
4.  Saat selesai, pastikan siswa menunjukkan layar **"Jawaban Tersimpan"**.

### C. Pengumpulan Jawaban (Pasca Ujian)
1.  Izinkan siswa menyalakan internet (atau tethering ke HP Anda).
2.  Minta siswa klik tombol **Upload** di aplikasi.
3.  Pastikan siswa mendapatkan **Kode Receipt** (Centang Hijau).
4.  Cek di Panel Admin: Jumlah upload yang masuk harus sama dengan jumlah siswa.

## 2. PANDUAN SISWA

### A. Persiapan (Di Rumah / Ada Internet)
1.  Buka Aplikasi Ujian.
2.  Login dengan akun Anda.
3.  Di Dashboard, cari ujian yang dijadwalkan.
4.  Klik tombol **DOWNLOAD**.
5.  Tunggu sampai status berubah jadi **"Siap Dikerjakan (Offline)"**.

### B. Mengerjakan Ujian (Di Kelas)
1.  **MATIKAN DATA / WIFI** (Wajib Mode Pesawat).
2.  Buka Aplikasi -> Klik **Mulai Ujian**.
3.  Kerjakan soal. Jawaban otomatis tersimpan (Autosave) setiap 10 detik.
4.  Jangan keluar aplikasi! Keluar aplikasi akan tercatat sebagai pelanggaran.
5.  Jika sudah selesai, klik **SELESAI** -> **KUNCI JAWABAN**.

### C. Mengumpulkan (Upload)
1.  Setelah jawaban dikunci, nyalakan kembali Data/Wi-Fi.
2.  Klik tombol **UPLOAD JAWABAN**.
3.  Tunggu sampai muncul **Centang Hijau** dan **Kode Bukti**.
4.  Tunjukkan Kode Bukti ke pengawas sebelum keluar ruangan.
